## Summary

- Adds dedicated `Problemas` sheet output focused on actionable rows (`pendente` and `divergente`) in reconciliation XLSX export.
- Keeps a clear fallback line when no pending/divergent rows exist, so the sheet is always present and readable.
- Adds API tests that open exported XLSX and validate the `Problemas` sheet structure/content for both with-issues and no-issues scenarios.
- Makes analysis/reconcile temp storage root path absolute from backend root to reduce runtime path ambiguity.

## Why

EPIC 5 activity 19 requires a specific report tab that highlights problems directly in the exported report, so accountants/operators can act faster on pending/divergent items without scanning the full detailed sheet.
