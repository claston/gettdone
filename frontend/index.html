FAQ

Perguntas frequentes sobre OFX Simples

O que é OFX Simples?

Um fluxo para converter extratos bancários em OFX de forma rápida, com revisão antes do download.

Quais formatos posso enviar?

Atualmente: PDF com texto.

Funciona para qualquer banco?

Funciona para os layouts suportados. Quando um layout não é reconhecido, você pode enviar para análise.

Preciso instalar algum programa?

Não. O uso é direto no navegador.

Quanto tempo leva a conversão?

Normalmente segundos, variando conforme tamanho e complexidade do arquivo enviado.

Posso revisar antes de baixar?

Sim. O fluxo inclui revisão de lançamentos antes da geração final.

O arquivo OFX gerado é pronto para importar?

Esse é o objetivo do fluxo: gerar OFX para importação direta no seu sistema.

O que fazer se o arquivo não converter?

Acione suporte com o exemplo do extrato para avaliação e melhoria de compatibilidade.
